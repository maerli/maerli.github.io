Equação de 2º grau 
    A equação de 2º grau é uma equação matemática que tem a seguinte forma geral:

ax² + bx + c = 0

Onde a, b e c são constantes e x é a variável. Essa equação é chamada de equação de 2º grau porque o maior expoente da variável é 2.

Para resolver uma equação de 2º grau, podemos usar a fórmula geral, que é dada por:

x = (-b ± √(b² - 4ac)) / 2a

Onde a, b e c são os coeficientes da equação e x é a solução da equação. O símbolo ± representa que devemos calcular a fórmula duas vezes, uma vez com o sinal + e outra vez com o sinal -. Isso nos dará as duas soluções da equação.

Por exemplo, considere a equação 2x² + 3x - 5 = 0. Para resolvê-la, basta substituir os valores de a, b e c na fórmula geral:

x = (-3 ± √(3² - 42(-5))) / 2*2

x = (-3 ± √(9 + 40)) / 4

x = (-3 ± √(49)) / 4

x = (-3 ± 7) / 4

Portanto, as soluções da equação são x = (-3 + 7) / 4 = 1 e x = (-3 - 7) / 4 = -2.
